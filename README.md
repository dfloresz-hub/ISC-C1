# **_Glosario ISC-C1_**

## 1) **API:**

Siglas que significan **"Application programming Interface"**, es una interfaz de programación de aplicaciones
usada en el desarrollo web y de software. Permite que las diferentes apps interactúen mediante solicitudes,
respuestas y compartan datos de manera segura y eficiente.

## 2) **Backend:**
Es un los sistema corporativo que se utilizan para dirigir una web o empresa, tales como sistemas de gestión
de pedidos, inventario y procesamiento de suministro.

## 3) **Backup:**


Es una copia de reserva que se hace de la información valiosa contenida en un sistema informático. Se realiza 
para evitar la pérdida de los datos originales debido a cualquier tipo de error o accidente, dado que un backup 
permite recuperarlos con facilidad.

## 4) **Cirfrado:**

En ciberseguridad es la conversión de datos de un formato legible a un formato codificado. Los datos cifrados 
solo se pueden leer o procesar luego de descifrarlos.

## 5) **Devops:**

 Es una combinación de prácticas, herramientas y una filosofía cultural que busca integrar los equipos de desarrollo 
 de software y operaciones.  El término proviene de la unión de las palabras development (desarrollo) y operations (operaciones), 
 reflejando su objetivo de eliminar barreras entre estos dos roles.

## 6) **Docker:**

Es una plataforma revolucionaria de código abierto que está cambiando la forma en que construimos, desplegamos y 
gestionamos el software.

## 7) **Embebido:**

Se refiere a algo que está contenido, absorbido o incorporado dentro de otra cosa, ya sea de manera física, abstracta o figurada.

## 8) **Firmware:**
 
Es un tipo de programa de software que permite controlar y comunicarse con el hardware de un equipo de forma directa y la gran 
mayoría de dispositivos electrónicos diferentes cuentan con firmware como un coche, un móvil, un rúter y ahora hasta lavadoras u otros 
electrodomésticos inteligentes de la casa.

## 9) **Frontend:**

Es la parte visible de un sitio web o aplicación con la que los usuarios interactúan directamente. Representa la interfaz gráfica que se 
ejecuta en el navegador del usuario y se encarga de mostrar elementos como el diseño, colores, botones, imágenes y animaciones.

## 10) **LaaS:**

Significa "Licencia como Servicio" (Licensing as a Service), es un modelo de servicio que permite a los clientes acceder a software y licencias de manera flexible y escalable.

## 11) **loT:**

Se refiere a una red de dispositivos físicos, vehículos y otros objetos físicos que están integrados con sensores, software y conectividad 
de red integrados, lo que les permite recopilar y compartir datos.

## 12) **Perifericos:**

En informática, aquellas piezas de hardware que no integran el núcleo del sistema computarizado sino que se conectan e interactúan de manera 
auxiliar con el sistema para permitir la transmisión de información hacia y desde el exterior.

## 13) **Pishing:**

Es un tipo de ciberataque que utiliza correos electrónicos, mensajes de texto, llamadas telefónicas o sitios web fraudulentos para engañar a 
la gente y hacer que comparta datos confidenciales, descargue malware o se exponga a la ciberdelincuencia.

## 14) **Transitores:**

Es un dispositivo electrónico semiconductor capaz de modificar una señal eléctrica de salida en respuesta a una de entrada. Puede funcionar como 
amplificador, conmutador, oscilador o rectificador de señales eléctricas.

## 15) **Uefi:**

es su sustituto, un sistema que ha remplazado a la BIOS y un nuevo estándar que tiene una serie de ventajas sobre este. Por eso hoy te explicamos 
qué es, como funciona o cómo puedes acceder desde Windows para configurarlo.
